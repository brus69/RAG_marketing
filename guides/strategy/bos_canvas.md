[1. НАЗНАЧЕНИЕ]
Построить стратегическую канаву для сравнения:
- Текущих игроков отрасли (конкуренты)
- Альтернативных решений (замена)
- Нового предложения (голубой океан)

[2. СТРУКТУРА КАНАВЫ]

2.1 Горизонтальная ось (Факторы конкуренции)
- Перечисли 5-7 ключевых факторов, на которых конкурирует отрасль
- Примеры категорий:
  • Цена / Доступность
  • Качество / Надёжность
  • Скорость / Удобство
  • Престиж / Статус
  • Функциональность / Возможности
  • Сервис / Поддержка
  • Эмоциональная ценность

2.2 Вертикальная ось (Уровень предложения)
- Шкала: Низкий (0) → Средний (5) → Высокий (10)
- Оцени каждый фактор для:
  a) Типичный игрок отрасли
  b) Альтернативное решение (замена)
  c) Новое предложение (гипотеза)

[3. ПРАВИЛА ЗАПОЛНЕНИЯ]

ДЕЛАЙ:
- Фокусируйся на 5-7 самых важных факторах (не больше!)
- Используй данные: отзывы, сайты конкурентов, исследования
- Сравнивай с альтернативами (не только с прямыми конкурентами)
- Ищи факторы, где все игроки на «одной волне»

НЕ ДЕЛАЙ:
- Не включай второстепенные факторы
- Не оценивай субъективно без данных
- Не игнорируй непрямых конкурентов (замены)

[4. ИНТЕРПРЕТАЦИЯ]

Признаки «Красного океана» (конкуренция):
- Все кривые похожи по форме
- Конкуренция идёт по одним и тем же факторам
- Различия только в уровне («лучше/дешевле»)

Признаки «Голубого океана» (новая ценность):
- Кривая нового предложения отличается по форме
- Некоторые факторы устранены или снижены
- Появились новые факторы, которых не было у конкурентов
- Фокус на других аспектах ценности

[5. ШАБЛОН ВЫВОДА]

Текущая отрасль фокусируется на:
1. {фактор_1} — уровень {0-10}
2. {фактор_2} — уровень {0-10}
...

Альтернативы конкурируют по:
1. {фактор_1} — уровень {0-10}
...

Новое предложение должно:
- Устранить: {факторы, которые не нужны}
- Снизить: {факторы ниже отраслевого стандарта}
- Повысить: {факторы выше отраслевого стандарта}
- Создать: {новые факторы, которых нет в отрасли}

[6. ПРИМЕР ЗАПОЛНЕНИЯ (абстрактный)]

Факторы: Цена, Скорость, Качество, Удобство, Престиж, Поддержка

Игрок А: [8, 6, 9, 7, 8, 9]
Игрок Б: [7, 7, 8, 8, 7, 8]
Альтернатива: [9, 5, 6, 9, 3, 5]
Новое: [6, 9, 7, 9, 5, 6]

Вывод: Новое предложение жертвует престижем и максимальной поддержкой, 
но выигрывает в скорости и удобстве при доступной цене.
